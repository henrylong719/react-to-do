# react-to-do (MERN)

use MongoDB, Express, React, Node

## Usage

change config.env file in config folder



```javascript

npm install

 cd client npm install
 cd ..
 
 # Run front and backend
 npm run dev
 
 # Backend only
 npm run server
 
 # Frontend only
 npm run client
 
 # Build client
 cd client
 npm run build
 
 ```
